CINEBRY adalah project web movie tracker ala Letterboxd yang dIbuat untuk mencatat dan mengelola daftar film favorit. Web ini dibangun memakai framework Laravel (dengan konsep MVC) dan Bootstrap 5 biar tampilannya langsung dapet vibe dark mode yang responsif.

Tema utama dari project ini adalah Pengelolaan Katalog Film. Fitur kuncinya ada di pengolahan array yang dinamis, di mana sistem bakal menampilkan daftar film yang berbeda (terpersonalisasi) tergantung dari user siapa yang lagi login.
